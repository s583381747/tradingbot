# QQQ EMA20 Trend Following Strategy

QQQ 1 分钟级别 EMA20 回调弹跳趋势跟随量化策略。2 年 Polygon SIP 数据回测 **PF=2.55、+33.26%**，经外部审计 + 微观结构压力测试验证。

---

## 策略概要（Plan F）

### 入场逻辑

```
1. 趋势确认    Close > EMA20 > EMA50（做多）或 Close < EMA20 < EMA50（做空）
2. 回调触碰    wick 在 EMA20 ± 0.15×ATR 范围内（几乎亲到均线）
3. 弹跳确认    下一根 bar 的 Close > 触碰 bar 的 High
4. 信号挂单    在触碰 bar 的 High + $0.05 挂 stop buy（3 bars 有效期）
5. 时间限制    不在 15:30 后新开仓，15:58 强平所有持仓
```

### 出场架构

```
Lock（20% 仓位）:
  - 固定止盈 0.3:1 R:R（约 $0.15-0.18）
  - 触发后止损移到保本 (breakeven)

Runner（80% 仓位）:
  - Chandelier 追踪止损: highest_high(40 bars) - 1.0×ATR
  - Lock 触发后开始追踪，只升不降
  - 无固定止盈上限，让利润奔跑
```

### 风控

```
仓位     每笔风险 = 1% 权益，最大仓位 = 25% 权益
日限     当日累计亏损 ≥ 2.5R 后停止交易至次日
佣金     IBKR $0.005/share
```

### 策略本质

高频小赌注 + 不对称收益分布：

| 类型 | 占比 | R 倍数 | 总 PnL 贡献 |
|------|------|--------|------------|
| 止损出场 | 20.2% | -1R | -$21,445 |
| 保本/微利（lock 赚但 runner 在 BE 出）| 67.7% | 0~0.1R | +$2,647 |
| Runner 中等盈利 | 9.6% | 1R~5R | +$35,590 |
| Runner 大赢 | 2.5% | 5R+ | +$16,462 |

**2.5% 的大赢交易贡献了 ~50% 的总利润。**

### 4. 入场信号验证

## 关键技术决策

| 决策 | 原因 |
|------|------|
| **IEX → Polygon SIP** | IEX 有 13% zero-range bars (H=L)，volume 只有 SIP 的 2% |
| **佣金 0.1% → $0.005/share** | 旧设置 = $0.50/share（真实 IBKR 的 100 倍），是 Gen 0-15 全亏的根因 |
| **backtrader → pandas** | backtrader 的 Stop order 有 1 bar fill 延迟，交易数从 5000 降到 850 |
| **分仓出场** | 不分仓 PF=1.13，分仓 PF=2.55（lock 提供现金流，runner 捕捉趋势） |
| **Touch tolerance 1.0 → 0.15 ATR** | 宽容差让太远的信号入场，R:R 全面下降 ~10pp |
| **Chandelier 40/1.0** | 73 种 trail 配置测试中最优，比 EMA trail 更稳定 |

---

## 已知风险

1. **BE 滑点是真实成本** — 64% 交易在保本出场，实盘每笔约亏 $0.02-0.05。压力测试显示 -11% PF。
2. **执行延迟敏感** — 需低延迟 VPS (AWS us-east-1)，不能从本地跑。
3. **全自动化必须** — 日均 ~9 笔交易，手动不可行。
4. **容量限制** — 1 分钟 scalping，适合 $100K 级别，不适合大资金。
5. **单标的风险** — 仅在 QQQ 验证，SPY 交叉验证待做。
